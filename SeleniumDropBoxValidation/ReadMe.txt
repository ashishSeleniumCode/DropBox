The Project has three class and two config files mentioned under,

1. BaseTest.cs:
Contains two methods which handles the open and closing of browser once the script execution is completed.

2. PageObjects.cs: 
Contains all the webelements.

3. TestClass.cs: 
The test is divided into a order and assertion is applied in this class.

4. Environment config:
All the parameters used in scripts can be updated from this file for ex: url, username, Path to upload file etc
